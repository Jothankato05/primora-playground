# Language Basics

Learn the core syntax and semantics of Primora.

## Variables
```primora
x = 10
y = "hello"
```

## Data Types
- `int`, `float`, `string`, `bool`, `list`, `dict`

## Operators
- Arithmetic: `+`, `-`, `*`, `/`, `%`
- Comparison: `==`, `!=`, `>`, `<`, `>=`, `<=`
- Logical: `and`, `or`, `not`

## Control Flow
```primora
if (x > 0) {
    echo "Positive"
} else {
    echo "Non-positive"
}
```

## Functions
```primora
def add(a, b) {
    return a + b
}
result = add(2, 3)
```

## Comments
```primora
# This is a comment
```

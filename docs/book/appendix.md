# Appendix

## Primora Syntax Reference
- Variable assignment: `x = 1`
- Function definition: `def foo() { ... }`
- Conditionals: `if (cond) { ... } else { ... }`
- Loops: `for x in list { ... }`
- Comments: `# comment`

## Built-in Functions
- `echo`, `plot`, `summarize`, `translate`, `fetch`, `search`, `predict`, `email`, `automate`

## Reserved Words
- `if`, `else`, `def`, `return`, `try`, `catch`, `for`, `in`, ...
